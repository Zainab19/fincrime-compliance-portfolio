# Financial Crime Compliance Portfolio
A self-made Excel simulation covering core entry-level 
financial crime compliance workflows.
Built to demonstrate practical understanding of AML, KYC, 
and Sanctions concepts that come under financial crime compliance.

## What's Inside
### Sheet 1 — KYC Customer Risk Scoring Model
- 14-factor weighted scoring model based on FATF Recommendation 10
- Risk factors cover: customer type, PEP status, geography, 
  products, transaction behaviour, and source of funds
- Auto-calculates Low / Medium / High / Very High risk rating
- Includes EDD trigger thresholds and risk scale reference
### Sheet 2 — AML Transaction Monitoring Log
- 10 fictional customer transactions screened against 3 rules:
  structuring detection, high-value alerts, round-amount flags
- Each alert includes analyst disposition and documented reasoning
- Covers SAR filing decisions, false positive closures, 
  EDD escalations, and MLRO referrals
- Summary metrics include SAR rate and false positive rate
### Sheet 3 — Sanctions and PEP Screening Log
- 10 screening cases across OFAC SDN, UN Consolidated List, 
  EU Financial Sanctions, and PEP databases
- Demonstrates fuzzy name matching logic and match type analysis
- Includes true match, possible match, and false positive cases
- Each case has documented analyst reasoning and action taken

## Frameworks Referenced
- FATF Recommendations (especially Rec. 10 on CDD)
- Risk-Based Approach (RBA)
- 3 Lines of Defence model
- PMLA and FIU-IND reporting requirements (India)
- OFAC SDN screening methodology
- EU 6th Anti-Money Laundering Directive (6AMLD)

## Tools Used
- Microsoft Excel (formulas, conditional logic, structured data)
